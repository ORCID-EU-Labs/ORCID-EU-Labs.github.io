Datacite/ORCID search & claim tool

Tool for searching for works in the DataCite metadata catalogue and adding them to an ORCID profile. Based on the code for CrossRef Metadata Search.

Status: in production at http://datacite.labs.orcid-eu.org

Code: https://github.com/ORCID-EU-Labs/cr-search

ISNI2ORCID

Tool for searching the ISNI registry by name and adding one or more ISNIs to ORCID profile as as external identifiers.

Status: in beta testing at http://isni2orcid.labs.orcid-eu.org

Code: https://github.com/gthorisson/isni2orcid

ORCID Feed

ORCID works as machine-readable feeds in RSS, JSON and other formats.

Status: in beta testing at http://feed.labs.orcid-eu.org

Code: https://github.com/ORCID-EU-Labs/orcid-feed

All current Labs projects are being undertaken as part of the ODIN project - ORCID and DataCite Interoperability Network.
